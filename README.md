# django_blogging

### About project:
Simple blogging application with Django. A back-end technical task for Bitpin.

### Run project:
```
docker-compose run
```
After running, project will be served at ```http://127.0.0.1:8000/```. Also admin panel is available at ```http://127.0.0.1:8000/admin/```

### Run tests:
```
./run_tests.sh
```
